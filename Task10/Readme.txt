## Purpose
CIFAR-10 데이터셋을 사용해 VGG-16과 ResNet-50 모델을 학습하고
테스트 정확도를 평가하는 코드입니다.

================================================================
## Files
- main.py          : 데이터 로드, 학습, 테스트 실행 메인 코드
- resnet50_full.py : ResNet-50 모델 구현 (ResidualBlock, ResNet50_layer4)
- vgg16_full.py    : VGG-16 모델 구현 (VGG, make_layers, vgg16)

================================================================

## How to Run

1. 체크포인트 파일을 main.py와 같은 디렉토리에 위치시킵니다.
   - resnet50_epoch285.ckpt
   - vgg16_epoch250.ckpt

2. main.py에서 사용할 모델을 선택합니다. (기본값: ResNet-50)

   [ResNet-50 사용 시]
   model = ResNet50_layer4().to(device)
   PATH = 'resnet50_epoch285.ckpt'

   [VGG-16 사용 시]
   아래 두 줄의 주석을 해제하고 ResNet-50 관련 두 줄을 주석 처리합니다.
   # model = vgg16().to(device)
   # PATH = 'vgg16_epoch250.ckpt'

3. 실행합니다.
   > python main.py

================================================================
## Parameters

| 파라미터        | 기본값  | 설명              |
|----------------|---------|-------------------|
| num_epochs     | 1       | 학습 epoch 수     |
| learning_rate  | 0.001   | 초기 학습률       |
| batch_size     | 100     | 미니배치 크기     |

================================================================
## Expected Results

- VGG-16    : Test Accuracy 약 85% 이상 (실험 결과: 86.70%)
- ResNet-50 : Test Accuracy 약 80% 이상 (실험 결과: 81.37%)

## Notes

- CIFAR-10 데이터셋은 첫 실행 시 자동으로 다운로드됩니다.
  (download=True로 설정되어 있음)
- GPU 사용 가능 환경에서는 자동으로 CUDA를 사용합니다.
- 학습 완료 후 모델이 resnet50_final.ckpt 또는 vgg16_final.ckpt로
