1. adaptivethreshold.cpp
===========================================================

[목적]
  - Moving Average를 이용한 Adaptive Thresholding구현
  - 조명이 불균일한 문서 스캔 이미지에서 텍스트를 효과적으로 분리합니다.
  - 각 픽셀마다 주변 지역의 평균값을 임계값으로 사용합니다.

[입력 / 출력]
  - 입력 : writing.jpg (문서 스캔 이미지, 그레이스케일로 변환하여 사용)
  - 출력 : 이진화된 이미지 (픽셀값 0 또는 255)

[결과 창]
  - "Grayscale"          : 입력 그레이스케일 이미지
  - "Adaptive_threshold" : 적응형 이진화 결과 이미지

[파라미터 설명]
  - N (필터 커널 크기)
      커널 크기는 (2N+1) x (2N+1)로 결정됩니다.
      N이 클수록 더 넓은 영역의 평균을 사용합니다.
      기본값: N = 2 → 5x5 커널

  - b (가중치)
      임계값 T = b x 지역평균
      b가 1보다 작을수록 더 많은 픽셀이 흰색(255)으로 분류됩니다.
      기본값: b = 0.9

[파라미터 변경 방법]
  main() 함수 내 아래 줄에서 수정:

      output = adaptive_thres(input_gray, 2, 0.9);
                                          ^  ^^^
                                          N   b

2. kmeans.cpp
===========================================================

[목적]
  - OpenCV의 cv::kmeans 함수를 이용한 이미지 세그멘테이션 구현
  - 그레이스케일 및 컬러 이미지에 대해 4가지 방식으로 클러스터링을 수행합니다.

[입력 / 출력]
  - 입력 : lena.jpg
  - 출력 : 클러스터링된 세그멘테이션 이미지 (레이블 맵)

[결과 창 - 총 6개]
  - "Original"                    : 원본 컬러 이미지
  - "Original Gray"               : 원본 그레이스케일 이미지
  - "Grayscale Intensity"         : 밝기값만으로 클러스터링한 결과 (그레이스케일)
  - "Grayscale Intensity & Position" : 밝기값 + 위치로 클러스터링한 결과 (그레이스케일)
  - "clustered image"             : RGB 색상만으로 클러스터링한 결과 (컬러)
  - "RGB Intensity & Position 5D" : RGB 색상 + 위치(5D)로 클러스터링한 결과 (컬러)

[파라미터 설명]
  - clusterCount (클러스터 수 K)
      이미지를 몇 개의 영역으로 나눌지 결정합니다.
      기본값: 10

  - attempts (반복 횟수)
      서로 다른 초기값으로 kmeans를 실행하는 횟수입니다.
      횟수가 많을수록 결과가 안정적이지만 느려집니다.
      기본값: 5



[파라미터 변경 방법]
  - K 변경:
      int clusterCount = 10;   

  - attempts 변경:
      int attempts = 5;

3. meanshift.cpp
===========================================================

[목적]
  - OpenCV의 pyrMeanShiftFiltering 함수를 이용한 Mean Shift 세그멘테이션 구현
  - 그레이스케일 및 컬러 이미지에 각각 Mean Shift 필터를 적용합니다.
  - 주의: cv::cuda::meanShiftSegmentation은 정상 동작하지 않아
          pyrMeanShiftFiltering을 대신 사용합니다.

[입력 / 출력]
  - 입력 : lena.jpg
  - 출력 : Mean Shift 필터링된 이미지 (그레이스케일 / 컬러)

[결과 창 - 총 4개]
  - "Original"      : 원본 컬러 이미지
  - "Original Gray" : 원본 그레이스케일 이미지 (3채널 변환 후 표시)
  - "Meanshift Gray": 그레이스케일 이미지에 Mean Shift 적용 결과
  - "Meanshift"     : 컬러 이미지에 Mean Shift 적용 결과

[파라미터 설명]
  pyrMeanShiftFiltering(입력, 출력, sp, sr, maxLevel)

  - sp (Spatial Window Radius, 공간 윈도우 반경)
      주변 몇 픽셀까지 같은 영역으로 볼지 결정합니다.
      값이 클수록 더 넓은 영역이 하나로 합쳐집니다.
      기본값: 31

  - sr (Color Window Radius, 색상 윈도우 반경)
      색상 차이가 얼마나 나도 같은 영역으로 볼지 결정합니다.
      값이 클수록 더 많은 색상이 하나로 합쳐집니다.
      기본값: 20

  - maxLevel (피라미드 레벨)
      이미지 피라미드의 단계 수입니다.
      값이 클수록 처리 속도는 빠르지만 결과가 거칠어집니다.
      기본값: 그레이스케일=1, 컬러=3

[파라미터 변경 방법]
  main() 함수 내 아래 줄에서 수정:

      pyrMeanShiftFiltering(input_gray, output_gray, 31, 20, 1);
                                                     ^^  ^^  ^
                                                     sp  sr  maxLevel

      pyrMeanShiftFiltering(input, output, 31, 20, 3);
                                           ^^  ^^  ^
                                           sp  sr  maxLevel
